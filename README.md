# hashtag-pandas
Pandas para Python: Guia DEFINITIVO do Básico ao Avançado
Guia publicado no Youtube: https://www.youtube.com/watch?v=TMfMkgLkeBQ

## Passo-a-passo

1. Crie e ative um ambiente virtual conforme abaixo:
- No Linux
```bash
python -m venv .venv
source .venv/bin/activate
```
- No Windows
```bash
python -m venv .venv
.\.venv/bin/activate
```

2. Instale as dependências
```bash
pip install -r requirements.txt
```

3. Instale a extensão 'Jupyter' do VSCode
4. Crie um arquivo com o nome analise.ipynb
